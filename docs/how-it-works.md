# How It Works - OpenClaw Launch

OpenClaw Launch helps teams choose a model, bind a channel, pay for a plan, track provisioning, and operate OpenClaw instances from a hosted console.

## Choose the launch shape

Select the channel, model route, plan, and expected assistant use case before collecting any sensitive setup data.

## Start checkout

Create the order and payment session while keeping the product page and plan context available.

## Provision and bind

Use payment status and submitted setup details to create or update the OpenClaw instance and channel binding.

## Operate from console

Return to the console for launch state, additional OpenClaws, upgrades, and support notes.

## Pricing and Support

Plans cover deployment capacity, console access, channel setup, runtime tracking, upgrade help, and support through the public pricing flow.
